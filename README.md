# İşletim Sistemleri 10. Hafta Ödevi

Bu repository, İşletim Sistemleri dersi için 10. hafta kapsamında
yapılan ödev çalışmalarını içermektedir.

## İçerik

Bu ödevde aşağıdaki konularla ilgili basit Python uygulamaları yapılmıştır:
- Thread kullanılarak çoklu programlama örneği
- Process kullanılarak çoklu işlemci örneği
- Thread ve process arasındaki farkın gösterilmesi
- Amdahl Yasasına göre hızlanma hesaplama programı

## Amaç

Bu ödevin amacı, çok çekirdekli sistemlerde
eşzamanlılık ve paralellik kavramlarını
uygulama üzerinden daha iyi anlamaktır.
